# ChatGPT Queue
Simple userscript to queue messages to ChatGPT. Basically, it's https://chromewebstore.google.com/detail/chatgpt-queue-save-time-w/iabnajjakkfbclflgaghociafnjclbem, but as a userscript so can be used in browsers other than Chrome (and free).

## How to download
1. Install a userscript manager like [Tampermonkey](https://www.tampermonkey.net/) or [Violentmonkey](https://violentmonkey.github.io/get-it/) (I prefer Violentmonkey).
2. Install the userscript from [here](https://github.com/HailXD/chatgpt-queue/raw/refs/heads/main/script.user.js). If extension is installed correctly, it should detect the userscript and prompt you to install it, otherwise, just copy-paste the code from `script.user.js` into a new userscript in your userscript manager.
3. Done!
4. The extension auto updates to this github by default, but you can disable if from the extension if you want.
