# ChatGPT Queue
Simple userscript to queue messages to ChatGPT. Basically, it's https://chromewebstore.google.com/detail/chatgpt-queue-save-time-w/iabnajjakkfbclflgaghociafnjclbem, but as a userscript so can be used in browsers other than Chrome (and free).

## How to download
1. Install a userscript manager like [Tampermonkey](https://www.tampermonkey.net/) or [Violentmonkey](https://violentmonkey.github.io/get-it/).
2. Install the userscript
3. Done!
